# Our Travis Country

## About
A web application built with Next.js, Supabase, and Mailgun, **Our Travis Country** is a community engagement platform and bylaw archive for residents of Travis Country to provide awareness and transparency regarding the cummunity's HOA bylaws.

## Technologies
- **Next.js** - Frontend framework utilizing React, Typscript, and Tailwind CSS.
- **Supabase** - Provides a postgreSQL database to store site content and handles authentication
- **Mailgun** - Email API for sending auth and outreach emails.

## Features
- User authentication
- Admin dashboard for managing site content
- Email messaging
- Responsive design
